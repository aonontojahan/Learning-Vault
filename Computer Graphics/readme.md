## Overview
**Computer Graphics** is the field of visual computing concerned with generating images and art with computers. It involves the creation, manipulation, and representation of visual data using algorithms and hardware.

### Brief History
- **1960s**: Early work by Ivan Sutherland (Sketchpad) and others laid foundations for interactive graphics.
- **1980s**: Rise of raster graphics, Pixar’s RenderMan, and films like *Tron*.
- **1990s-Present**: Real-time graphics (OpenGL/DirectX), GPUs, ray tracing, and AI-driven graphics (e.g., Stable Diffusion).

Applications span gaming, film (CGI in movies like *Avatar*), simulations, VR/AR, medical imaging, CAD, and data visualization.

## Key Concepts
- **2D vs 3D Graphics**: 2D (pixels, vectors); 3D (vertices, meshes, scenes).
- **Rasterization vs Ray Tracing**: Rasterization (real-time, GPUs); Ray tracing (photorealistic, path tracing).
- **Coordinate Systems**: World, view, projection matrices for transformations (rotation, scaling, translation).
- **Graphics Pipeline**: Vertex processing → Rasterization → Fragment shading → Output.

## Rendering Techniques
| Technique | Description | Use Cases |
|-----------|-------------|-----------|
| Phong Shading | Local illumination model (ambient, diffuse, specular). | Basic 3D games. |
| Physically Based Rendering (PBR) | Realistic materials using BRDFs. | Modern games (Unreal Engine). |
| Global Illumination | Indirect lighting (radiosity, path tracing). | Offline rendering (Blender Cycles). |
| Tessellation | Dynamic subdivision of geometry. | High-detail models. |
| Anti-Aliasing | Smooth edges (MSAA, FXAA, TAA). | All real-time rendering. |

## Tools and Libraries
- **APIs**: OpenGL, Vulkan, DirectX 12, Metal.
- **Game Engines**: Unity, Unreal Engine, Godot.
- **Modeling/Animation**: Blender, Maya, 3ds Max.
- **Programming**: C++ (performance), Python (prototyping with Pygame/Matplotlib), WebGL (browser-based).
- **GPU Compute**: CUDA, OpenCL for simulations.

## Applications
- **Entertainment**: Video games, movies, animations.
- **Scientific Visualization**: Atom/molecular simulations, fluid dynamics.
- **Design/Engineering**: CAD (AutoCAD), architecture walkthroughs.
- **Emerging**: Metaverse, generative AI art, real-time ray tracing (NVIDIA RTX).
